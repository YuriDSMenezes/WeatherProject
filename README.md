<h2 id="use">Getting started</h2>
```bash
# Clone this repository
$ git clone https://github.com/YuriDSMenezes/WeatherProject

# Go into the repository
$ cd react-rocketshoes

# Install dependencies
$ yarn install

# Run the app
$ yarn start
```
